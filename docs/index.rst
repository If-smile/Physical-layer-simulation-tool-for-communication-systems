pyberlab documentation
======================

``pyberlab`` is a reproducible Python laboratory for bit-error-rate experiments
with digital modulation schemes. It provides unit-power constellations,
memoryless channel models, theoretical BER baselines, adaptive Monte Carlo
simulation, CSV export, and plotting helpers.
